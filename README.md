### 1. Мета роботи
Навчитися створювати графічні інтерфейси користувача (GUI) мовою Java за допомогою стандартного пакету `javax.swing`. Ознайомитися з роботою контейнерів `JFrame`, компонентів вибору `JComboBox`, текстових зон `JTextArea`, механізмами розташування `BorderLayout`/`FlowLayout` та обробкою подій натискання кнопок за допомогою `ActionListener`.

---

### 2. Виконання вимог лабораторної роботи

* **На "трійку":** Створено базовий Swing-інтерфейс, підключено зовнішній файл-бібліотеку `MyBank.jar` через `Properties -> Libraries` у NetBeans. Додано базову структуру вікна та кнопку відображення інформації.
* **На "чотири":** Реалізовано динамічне зчитування даних про клієнтів банку та їхні фінансові рахунки з бінарного файлу даних `data/test.dat` (використовуючи напрацювання ЛР №8). Переписано виведення — при обранні клієнта у `JComboBox` та натисканні **Show** у текстовому полі відображаються деталі абсолютно **усіх** його рахунків.
* **На "п'ять":** Додано нову функціональну кнопку **Report**, яка при натисканні генерує та відображає зведений фінансовий звіт по всій базі даних банку (аналогічно роботі `CustomerReport`).
